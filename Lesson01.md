# Подсказки покомандной строке

Команда смены директории
```sh
cd ~/Desktop/Git
```

Команда отображения текущей директории.
```sh
pwd
```

Листинг текущей директории Windows:
```sh
dir
```
Linux, MacOS:
```sh
ls
```

Удаление файла Windows:
```sh
del <fileName>
```
в Linux, MacOS:
```sh
rm <fileName>
```
